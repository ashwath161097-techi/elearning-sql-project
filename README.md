# E-Learning Platform Purchases Analysis using MySQL

## Project Description

This project analyses purchase data from an online learning platform using MySQL.

The database contains three tables:

- Learners
- Courses
- Purchases

## Tools Used

- MySQL
- SQL

## Features

- Database creation
- Table relationships
- Joins
- Analytical queries
- Revenue analysis
